Idep - Sistema academico (Gestión de notas)
Este proyecto es una aplicacion web inicial, para la gestion academica de un colegio, implementa un modulo de autentificación usando Javascript, esta
es una fase inicial, del area de frontend

Funciones

 Solicita al usuario el ingreso con el correo  de usuario y la contraseña,  Lee y  muestra los datos, Verifica sin los datos ingresados son correctos,permite el acceso cuando los datos sean válidos, Indica cuando los datos son incorrectos, controla intentos de acceso muestra un mensaje de bloqueo cuando se superen tres intentos fallidos.

Instrucciones de Uso

Abre el archivo index.html
Ingresa las siguientes credenciales de prueba:Correo: usuario@Idep.edu
